# GitHub Pages Configuration

This directory contains the static build of the Cafe Bar website optimized for GitHub Pages.

## Deployment

The contents of this folder are ready to be deployed to GitHub Pages. To deploy:

1. Push this repository to GitHub
2. In your GitHub repository settings, enable GitHub Pages
3. Select the `github` folder as the source
4. Choose the root directory as the deployment folder

## Files Structure

- `index.html` - Main homepage
- `about/` - About page
- `menu/` - Menu page  
- `gallery/` - Gallery page
- `contact/` - Contact page
- `images/` - All image assets
- `_next/` - Next.js static assets
- `robots.txt` - SEO robots file
- `sitemap.xml` - SEO sitemap

## Notes

- All assets are optimized for static hosting
- Images are unoptimized as required by static export
- Routes include trailing slashes for proper GitHub Pages routing
