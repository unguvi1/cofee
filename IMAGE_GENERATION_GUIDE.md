## Design Guidelines for All Images

### Style Consistency:
- **Warm color palette**: Amber, brown, cream tones
- **Lighting**: Soft, warm, natural lighting preferred
- **Composition**: Clean, professional, slightly artistic
- **Quality**: High resolution, photorealistic
- **Mood**: Inviting, cozy, professional

### Technical Specs:
- **Aspect ratio**: 16:9 for hero, 1:1 for gallery, 4:3 for food items
- **Resolution**: Minimum 1920x1080 for hero, 1024x1024 for others
- **File format**: JPEG for photos, PNG for transparent logos

## Batch Generation Tips

### Using Microsoft Designer:
1. Sign in with Microsoft account
2. Use "Define your own" option
3. Copy-paste the prompts above
4. Add "4K quality, professional photography" for better results
5. Download in highest resolution available

### Using Leonardo.Ai:
1. Choose "Photography" style
2. Use "Realistic Vision" model
3. Set aspect ratio to match requirements
4. Generate multiple versions to choose best

### Quick Prompt Template:
```
[Subject], [Style], [Lighting], [Composition], [Quality], [Background]
```

Example:
```
Cappuccino with latte art, professional food photography, soft natural lighting, top-down close-up, 4K ultra realistic, minimalist white background
```
